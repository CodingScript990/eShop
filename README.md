<h3>[기술 스택]</h3>
<ul>
 <li>React</li>
 <li>JavaScript</li>
 <li>Fetch API</li>
 <li>Sanity[React CMS => Platform]</li>
 <li>Tailwindcss[CSS => Fram-Work]</li>
 <li>React-google-login[Package]</li>
 <li>React-icons[Package]</li>
 <li>React-router-dom@v6</li>
 <li>Uuidv4</li>
</ul>
<br/>
<h3>[구현 기능]</h3>
 - 계정
<ul>
  <li>소셜 로그인[Google], 로그아웃</li>
</ul> 
 - 제품 
 <ul>
  <li>판매제품 업로드[CRD]페이지</li>
  <li>판매제품 디테일 페이지</li>
  <li>찜하기</li>
  <li>물품 검색하기</li>
  <li>카테고리별 정렬기능</li>
</ul>
 - 유저
<ul>
  <li>클라이언트 페이지</li>
  <li>제품에 댓글 기능</li>
</ul>
